Open the program.

Either copy SVG path data into profile 1 and profile 2 or use the choose file option to open an existing SVG file from your computer.

The program will automatically create the intersection, which can be downloaded with the download STL file button on the right side.

You can preview the STL file with the window on the bottom right side of the screen. You can inspect the preview by modifying the parameters on the top right side of the screen.

Viewing angle adjusts the angle you are viewing the 3D model from.

Rotation speed sets the rate at which the model spins.

Perspective distance changes the model looks from different distances without changing the size of the object.
